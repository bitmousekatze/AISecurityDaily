Prompts Become Shells: Microsoft Semantic Kernel CVEs Turn Prompt Injection into Host RCE

CVE-2026-25592 & CVE-2026-26030Critical


Microsoft researchers disclosed two critical vulnerabilities in Semantic Kernel, a widely used open-source AI agent orchestration framework. A crafted prompt injection can now escalate to full remote code execution on the host running the agent.


The flaws exploit how agents handle untrusted inputs in vector stores and tool-calling paths. This is the clearest real-world proof yet that prompt injection is no longer “just a chatbot problem” — it’s a code execution vector in production agent frameworks.

Why It Matters for Vibe Coders
If you’re using Semantic Kernel, LangChain, or any agent framework with tool access, your agents can now be turned into remote shells with a single malicious prompt. The execution boundary between “reasoning” and “doing” has officially vanished.
Action Items
  • Update Semantic Kernel immediately and review all agent deployments.
  • Add strict input sanitization + output validation before any tool execution or code generation.
  • Run agents in isolated sandboxes with minimal privileges (no host filesystem/shell access unless absolutely required).
  • Test your agents against prompt-to-RCE payloads (Microsoft published PoCs).

Active · Browser Agents Via LayerX Security · Ongoing Reports

ClaudeBleed Chrome Extension Hijack — Remote Prompt Injection Still Live

The Claude Chrome extension vulnerability (ClaudeBleed) continues to see active discussion and PoC sharing. Any other extension (or malicious site) can inject prompts and hijack the agent to steal data from Gmail, GitHub, Drive, etc.


No user interaction required. This proves browser-based AI agents are now prime real estate for remote takeovers.

Why It Matters for Vibe Coders
Your daily AI coding workflow in the browser is one compromised extension or tab away from full workspace compromise. The “helpful sidebar” just became an always-on attack surface.
Action Items
  • Disable or isolate the Claude (and any AI) Chrome extension until fully patched.
  • Audit all browser extensions with AI/agent capabilities.
  • Use sandboxed browser profiles or containers for AI tooling.
